# مدير الكتب بالذكاء الاصطناعي - Laravel

## نظرة عامة

نظام متكامل لإدارة ومعالجة الكتب باستخدام الذكاء الاصطناعي. يتيح النظام استخراج النصوص من ملفات PDF، تحسين المحتوى، الترجمة، التلخيص، وإنشاء مقالات احترافية للمدونة.

## الميزات الرئيسية

### 1. معالجة ملفات PDF
- استخراج النصوص من ملفات PDF بدقة عالية
- الحفاظ على التنسيق الأصلي
- دعم الملفات الكبيرة (حتى 50 ميجابايت)

### 2. معالجة الذكاء الاصطناعي
- **تحسين النصوص**: تحسين جودة النصوص وتنظيمها
- **ترجمة النصوص**: ترجمة إلى 16 لغة مختلفة
- **تلخيص النصوص**: إنشاء ملخصات مختصرة ومفيدة
- **تلخيص على هيئة نقاط**: تنظيم المحتوى في نقاط واضحة
- **استخراج معلومات الكتاب**: استخراج العنوان والمؤلف والملخص
- **إنشاء مقالات المدونة**: كتابة مقالات احترافية للمدونة

### 3. إدارة المحتوى
- **مقالات المدونة**: إنشاء وإدارة مقالات احترافية
- **أنواع المقالات**: مقالات مدونة، مراجعات كتب، ملخصات، تحليلات، أدلة دراسية
- **إدارة الحالة**: مسودة، منشور، مؤرشف
- **الكلمات المفتاحية للـ SEO**: استخراج تلقائي للكلمات المفتاحية
- **إحصائيات المقالات**: عدد الكلمات، تاريخ المعالجة

### 4. إدارة الملفات
- رفع ملفات PDF
- استخراج النصوص
- تنظيم الملفات في مجلدات
- دمج وتقسيم الملفات
- إعادة ترتيب الفصول

### 5. دعم متعدد اللغات
- العربية والإنجليزية والفرنسية والإسبانية
- الألمانية والإيطالية والبرتغالية والروسية
- الصينية واليابانية والكورية والتركية
- الفارسية والأردية والهندية والبنغالية

## التثبيت والإعداد

### المتطلبات
- PHP 8.2+
- Composer
- MySQL 8.0+
- Node.js 18+

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/your-repo/laravel-ai-books-manager.git
cd laravel-ai-books-manager
```

2. **تثبيت التبعيات**
```bash
composer install
npm install
```

3. **إعداد البيئة**
```bash
cp .env.example .env
php artisan key:generate
```

4. **تكوين قاعدة البيانات**
```bash
# تعديل ملف .env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ai_books_manager
DB_USERNAME=your_username
DB_PASSWORD=your_password
```

5. **تشغيل الهجرات والبذور**
```bash
php artisan migrate
php artisan db:seed
```

6. **بناء الأصول**
```bash
npm run build
```

7. **تشغيل الخادم**
```bash
php artisan serve
```

## إعداد Google Gemini AI

1. احصل على مفتاح API من [Google AI Studio](https://makersuite.google.com/app/apikey)
2. أضف المفتاح إلى ملف `.env`:
```
GOOGLE_GEMINI_API_KEY=your_api_key_here
```

## الاستخدام

### 1. رفع الكتب
- انتقل إلى "إدارة الملفات" > "رفع الملفات"
- اختر ملف PDF
- انتظر اكتمال استخراج النصوص

### 2. معالجة الذكاء الاصطناعي
- انتقل إلى "معالج الذكاء الاصطناعي"
- اختر الملفات المراد معالجتها
- اختر نوع المعالجة:
  - **استخراج معلومات الكتاب**: للحصول على العنوان والمؤلف والملخص
  - **تلخيص النص**: لإنشاء ملخص مختصر
  - **ترجمة النص**: لترجمة النص إلى لغة أخرى
  - **تحسين النص**: لتحسين جودة النص
  - **تلخيص على هيئة نقاط**: لتنظيم المحتوى في نقاط
  - **مقال احترافي للمدونة**: لإنشاء مقال مدونة احترافي

### 3. إدارة مقالات المدونة
- انتقل إلى "إدارة المحتوى" > "مقالات المدونة"
- عرض جميع المقالات المنشأة
- تعديل الحالة (مسودة/منشور/مؤرشف)
- تحرير المحتوى والكلمات المفتاحية
- إدارة أنواع المقالات المختلفة

### 4. إدارة النصوص المعالجة
- عرض النصوص المحسنة والمترجمة والملخصة
- تحميل النصوص كملفات نصية
- حذف النصوص غير المطلوبة

## هيكل قاعدة البيانات

### الجداول الرئيسية
- `books`: معلومات الكتب الأساسية
- `books_info`: معلومات تفصيلية عن الكتب
- `blog_articles`: مقالات المدونة
- `enhanced_texts`: النصوص المحسنة
- `translated_texts`: النصوص المترجمة
- `summarized_texts`: النصوص الملخصة
- `formatting_improved_texts`: النصوص المنسقة
- `processing_histories`: تاريخ المعالجات
- `ai_prompts`: توجيهات الذكاء الاصطناعي

## API Endpoints

### معالجة الذكاء الاصطناعي
- `POST /ai-processor`: معالجة النصوص بالذكاء الاصطناعي
- `GET /ai-processor/{bookId}`: عرض معالج الذكاء الاصطناعي لكتاب محدد

### تجربة الذكاء الاصطناعي
- `GET /ai-trial`: صفحة تجربة الذكاء الاصطناعي
- `POST /ai-trial/process`: معالجة نص تجريبي
